# Solana Wallet Tracker

Tracks most profitable Phantom wallets on Solana using pump.fun token data.

## Features
- Top Solana tokens by 24h volume
- Top pump.fun tokens by volume
- Top pump.fun creators by aggregated volume
- Trending new pairs

## Usage

```bash
python solana_wallet_tracker.py
```

## API
Uses [DexScreener API](https://docs.dexscreener.com/) for real-time data.
